โจทย์ : Floating point calculator
------------------------------------------------------------------------
รับค่าเลขทศนิยม (ตำแหน่งทศนิยมไม่เกิน 10 ตำแหน่ง) 3 ตัว โดย ตัวหากตัวที่ 3 เป็นผลบวกของ 2 ตัวแรกให้ output เป็น yes และหากไม่ใช่ ให้ output เป็น no 
------------------------------------------------------------------------
Input : 2 3 5
output: yes
Input : 0.3 0.4 0.7
output: yes
Input : 0.12 0.13 0.3
output: no
